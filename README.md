# marcolanconelli-cv

LaTeX source of my curriculum vitae, built with the
[AltaCV](https://github.com/liantze/AltaCV) class (v1.7.2).

**PDF:** [out/marcolanconelli-cv.pdf](out/marcolanconelli-cv.pdf)

## Stack

LaTeX · AltaCV 1.7.2 · pdflatex · biber

## Run

```bash
bash install.sh   # LaTeX dependencies (Ubuntu/Debian)
make              # -> out/marcolanconelli-cv.pdf
make clean        # drop auxiliary files, keep the PDF
make cleanall     # drop auxiliary files and the PDF
```
